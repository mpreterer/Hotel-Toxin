# MetaLamp Task 2 step
### Задане выполнено для компании MetaLamp
---
### Распаковка
Клонировать
>```git clone https://github.com/iammumhacker/Toxin_Hotel.git```

Установить
>```npm i```
---

Запустить webpack server
>```npm run dev```
---

Запустить production build
>```npm run build```
---
