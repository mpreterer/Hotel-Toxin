# MetaLamp Task 2 step
### Задане выполнено для компании MetaLamp
---
### Распаковка
Клонировать
>```git clone https://github.com/mpreterer/Hotel-Toxin.git```

Установить
>```npm i```
---

Запустить webpack server
>```npm run start```
---

Запустить production build
>```npm run build```
---
